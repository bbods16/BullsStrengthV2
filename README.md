# WeightRoom Platform (ForgeIQ)

An AI-powered multi-tenant SaaS platform for NCAA athletic performance monitoring and strength & conditioning management.

## Project Structure

- `apps/`: Frontend applications (Coach UI, Athlete Kiosk)
- `api/`: Backend API (Azure Functions)
- `packages/`: Shared TypeScript packages
- `database/`: SQL schema and migrations

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **API**: Azure Functions, TypeScript
- **Database**: Azure SQL
- **Validation**: Zod
- **Auth**: Microsoft Entra ID
